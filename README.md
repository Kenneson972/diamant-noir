# 💎 DIAMANT NOIR - Luxury Villa Concierge Platform

## 📋 Présentation
**Diamant Noir** est une solution logicielle haut de gamme destinée aux conciergeries de luxe. Elle combine une expérience client immersive inspirée d'Airbnb avec un centre de commandement assisté par Intelligence Artificielle.

## 🚀 Fonctionnalités Clés
- **Expérience Client Immersive** : Galeries plein écran, moteur de prix dynamique, et calendrier de disponibilité réel.
- **Directeur de Stratégie IA** : Assistant admin capable d'analyser les KPIs (RevPAR, occupation) et de détecter des anomalies business.
- **Magic Airbnb Import** : Création instantanée de villas à partir d'une simple URL Airbnb (détails + 50 photos HD).
- **Dashboard de Pilotage** : Gestion sécurisée du portfolio, registre des séjours, et carnet de maintenance logistique.
- **Synchronisation iCal** : Mise à jour automatique des calendriers avec Airbnb et Booking.com.

## 🛠️ Stack Technique
- **Framework** : Next.js 14 (App Router)
- **Base de données** : Supabase (PostgreSQL + Auth)
- **Paiements** : Stripe Checkout
- **Automatisations** : n8n Workflows + AI Agents (DeepSeek/OpenAI)
- **UI/UX** : Tailwind CSS + Shadcn/UI
